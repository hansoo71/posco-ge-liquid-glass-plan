# Design System

- Style: Liquid Glass, translucent panels, depth, aurora gradients.
- Palette: deep navy base, cyan/blue/lime highlights.
- Typography: Apple system / Pretendard / Noto Sans KR fallback.
- Layout: executive hero, KPI glass cards, agent cards, layered architecture, horizontal roadmap, detailed responsive document sections.
- Mobile: single-column cards, scrollable roadmap/tables, keep Korean line breaks readable.
